# 📊 Public Health Statistical Analysis Project

A data-driven public health analytics project that combines **descriptive statistics**, **inferential statistics**, **hypothesis testing**, and **data visualization** to uncover health-related insights from a synthetic healthcare dataset.

![Age vs BMI](screenshot.png)

## 📌 Project Overview

This project demonstrates the application of statistical methods to analyze public health data. The analysis explores relationships between demographic factors, lifestyle habits, and health conditions such as diabetes, hypertension, BMI, and cholesterol levels.

## 📂 Dataset Information

The dataset contains 500 healthcare records with demographic, lifestyle, and clinical variables including Age, BMI, Smoking Status, Diabetes, Hypertension, Cholesterol Level, and Glucose Level.

## 🎯 Objectives

- Understand inferential statistics concepts
- Perform hypothesis testing
- Analyze relationships between health indicators
- Evaluate smoking-related health outcomes
- Measure covariance and correlation
- Visualize healthcare trends

## 🔬 Analysis Performed

### Confidence Interval for Age
Estimated the population mean age using a 95% confidence interval.

### Independent T-Test
Compared BMI between smokers and non-smokers.

### Chi-Square Test
Evaluated association between smoking status and diabetes.

### ANOVA
Analyzed the impact of Age, Smoking Status, and Diabetes on BMI.

### Covariance & Correlation
Measured relationships between Age and BMI.

## 📈 Visualization

### Age vs BMI Scatter Plot

The scatter plot illustrates the relationship between age and BMI across the dataset.
